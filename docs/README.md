# taut.tk docs

Start with the project [README](../README.md) for the big-picture tour.

## Topics

- [Reactive primitives](reactive.md): signals, memos, effects, and owner cleanup.
- [Widgets](widgets.md): classic Tk, ttk, layout props, stacks, portals, and fragments.
- [Control flow](control.md): transparent `Show`, `For`, `Switch`, `Index`, `Dynamic`, and `ErrorBoundary` nodes.
- [Context](context.md): providers, typed context helpers, and forwarded children.
- [Stores](stores.md): immutable stores, lenses, mutable proxies, `produce`, `reconcile`, and `unwrap`.
- [Scheduling](scheduling.md): owned Tk event-loop callbacks and thread-to-UI dispatch.
- [Resources](resources.md): worker-thread loading, refreshes, mutation, and status.
- [Style](style.md): StyleX-ish style objects, merging, and component styling.
